# DrawRate

DrawRate is a web browser drawing gmae. I made it because I thought it would be interesting when you draw and system rates your drawing.

[Press this to play](https://markussendzelis.github.io/DrawRate/)

<img width="750" height="702" alt="Screenshot 2026-06-25 140239" src="https://github.com/user-attachments/assets/d5e3bae3-cd67-427e-88c9-f7f0b66d1ba4" />

## Features
- Drawing tools - pen, eraser, color paletter and bruhs size slider
- Timer - 60 seconds to draw
- Rating - rates your drawing based on strokes, converage and colors used
- 5 rounds - random words every round with different difficulties

## How to play
- Just draw whatever the word is
- Use different drawing tools
- Submit the drawing

## Tech stack
- JavaScript
- HTML
